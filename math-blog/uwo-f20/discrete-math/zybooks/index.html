Zybooks
Filed under discrete math, August 30, 2020.

I was going to write a long rant about how much i despise north american math textbooks but I’ll leave that for later. I’ll just say that if I had studied math using these books I’d probably never have decided to pursue it as a career.

I’m very glad that one of the casualties of this pandemic is going to be the calculus textbook (and its siblings). While planning for this course I came across several online textbooks that I plan to explore in detail sometime.

  • https://activecalculus.org/
  • http://discrete.openmathbooks.org/dmoi3.html
  • https://aimath.org/textbooks/approved-textbooks/

The above three links have free textbooks but I decided to not use them in a large class of 200 students as I might need some tech support. Instead, for this class I decided to use Zybooks.

Pros Cons
  • comprehensive list of topics
  • very flexible structure
  • animations embedded in each chapter
  • participation and challenge activities
  • easy to create textbook assignments
  • easy to export data
  • expensive, given that free textbooks exist
  • does not have proctored or timed tests
  • not sure how to integrate with Sage or Webwork

So far, I’m very happy with Zybooks. Let’s see how it works out.
